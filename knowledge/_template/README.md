# Knowledge Folder Template

Copy this folder and rename it for your domain (e.g., `payments/`, `marketplace/`, `analytics/`).

Fill in the markdown files below with your product's context. These are used by rules, agents, and skills to give you domain-aware assistance.

## Files

| File | Purpose |
|------|---------|
| `key-resources.md` | **Entry point** — links to all knowledge files, external docs, and a decision-making checklist |
| `strategy.md` | Strategic pillars, phased approach, quarterly priorities |
| `customer-segments.md` | Who you serve — personas, segments, pain points |
| `competitive-landscape.md` | Competitors, positioning, differentiation |
| `value-proposition.md` | Target value prop, product design principles |
| `metrics-and-targets.md` | Key metrics, definitions, financial targets |
| `key-learnings.md` | Recent experiment results, learnings, focus shifts |

**Start with `strategy.md` and `customer-segments.md`** — these have the biggest impact on skill quality.
